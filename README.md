This is my playground for GPU bare-metal interaction.
Currently its not possible to find any public documentation on how to manage GPU in bare metal level. The only thing I can practically use as a source of information is linux kernel drivers.
The goal of the project is to explore GPU capabilities without drive, and ultimately to create simpler learning material for deveopers who want to get into graphics driver programming.

My program is designed for AMD 1002 1638 Cezanne.
So far I was able to make a bootable executable that runs my program form flash drive and changes gpu registers to achieve some visual result. 
Now I'm working on next step: where to find meanings of all registers and how to run programs on gpu.
